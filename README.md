# [jimsegal.com](http://jimsegal.com)

## Running locally

Run a local webserver to not get blocked by browsers CORS policies:

```bash
  ./.dev-server
```

Getting `permission denied: ./.dev-server.sh`? Give it executable permissions:

```bash
  chmod +x .dev-server.sh
```
